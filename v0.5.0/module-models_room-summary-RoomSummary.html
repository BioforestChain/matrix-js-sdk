Class: RoomSummary

models/room-summary~RoomSummary

new RoomSummary(roomId, info)

Construct a new Room Summary. A summary can be used for display on a recent list, without having to load the entire room list into memory.
Parameters:
Name Type Description
roomId string Required. The ID of this room.
info Object Optional. The summary info. Additional keys are supported.
Properties
Name Type Description
title string The title of the room (e.g. m.room.name)
desc string The description of the room (e.g. m.room.topic)
numMembers Number The number of joined users.
aliases Array.<string> The list of aliases for this room.
timestamp Number The timestamp for this room.
